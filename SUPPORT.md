# Support

Process Bus Insight is a GPL-3.0-or-later public-beta engineering tool. Community support is maintainer based and best effort; no response-time, warranty, field-service, validation, or acceptance SLA is implied.

## Before opening an issue

- Confirm the latest post-transition public beta or current `main` build.
- Read `docs/QUICK_START.md` and `docs/TROUBLESHOOTING.md`.
- Reproduce with a known source and a supported Windows/Npcap capture path.
- Separate application behavior from publisher or external-device behavior by checking raw values or a sanitized capture where possible.

## Include this evidence

- application version or commit SHA;
- Windows and Npcap versions;
- adapter type and capture/replay path;
- selected SV/GOOSE/PTP target;
- expected and observed behavior;
- evidence source: automated test, replay, loopback, isolated laboratory equipment, or authorized commissioning environment; and
- the smallest legally shareable sanitized screenshot, SCL excerpt, log, or capture.

## Never post publicly

Customer, employer, station, project, credential, private address inventory, confidential capture, unsanitized SCL/device data, proprietary manual, support response, screenshot, report, binary, or extracted third-party material.

Security issues must use a private security advisory, not the public issue tracker.
