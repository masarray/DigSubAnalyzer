# Support

Process Bus Insight is an open-source public-beta engineering tool. Support is community and maintainer based; no response-time or field-service SLA is implied.

## Before opening an issue

- Confirm the latest public beta or current `main` build.
- Read `docs/QUICK_START.md` and `docs/TROUBLESHOOTING.md`.
- Reproduce with a known source and a supported Windows/Npcap capture path.
- Separate application behavior from publisher behavior by checking raw sample values or a sanitized capture where possible.

## Include this evidence

- Application version or commit SHA
- Windows and Npcap versions
- Adapter type and capture path
- Selected SV/GOOSE/PTP target
- Expected and actual behavior
- Minimal, sanitized steps
- Sanitized screenshot, SCL excerpt, log, or capture when legally shareable

## Never post publicly

Customer/site names, real project files, credentials, private IP/MAC inventories, confidential captures, or unsanitized SCL/device data.

Security issues must use a private security advisory, not the public issue tracker.
